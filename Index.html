TAGS DE CONTENIDO

Tags de contenido o elementos HTML, que consistene en un tag de apertura y un tag de cierre entre los cuales se mete contenido.
Los tags se pueden anidar, es decir se puede meter un tag dentro de otro, un ejemplo facil de ver es el salto de linea "<br>" que se puede anidar dentro de un parrafo.
A continuacion estan algunos de estos tags de contenido o elementos.

<!DOCTYPE html>

El <!DOCTYPE html> es aquel tag que indica al navegador que el documento que esta leyendo esta escrito en html. Es el tag padre del documento y no habra ningun otro tag sobre el, bajo el estara el tag <html>, vamos con el.

Tag <html>

El tag <html> es aquel tag de mas importancia por detras del DOCTYPE, y es el tag que define el elemento al completo, tendra tag de apertura y de cierre.
Este tag HTML puede tener dentro el atributo lang, que indicara al navegador en que idioma esta hecha la pagina, por ejemplo para saber de que idioma tiene que hacer una traduccion o para mostrar a una persona una pagina con un idioma acorde al que el usuario habla.
Dentro de este tag estaran otros dos tag mas que seran <head> y el <body>.

Tag <head>

El tag <head> es aquel que contendra el titulo de la pagina y tendra elementos que daran metadatos para el documento, uno de los tags que podremos tener dentro del <head> es el <title>.

Tag <title>

Es aquel Tag que indicara un titulo para la pagina, no on se vera en la propia pagina, sino en la pestaña de arriba.


Tag <meta>

Este tag aportara la informacion sobre el tipo de codificacion con la que esta hecha la pagina, normalmente usaremos el "UTF-8" y su sintaxis se ve asi:
<meta charset="UTF-8">

Tag <body>

Basicamente es el cuerpo de la pagina, donde va a estar practicamente todo el contenido y todos los tags

Parrafos <p>

El parrafo es simplemente eso, un texto, nada mas


Texto preformateado (Elemento <pre>)

Con este tag puedes escribir directamente con un formato distinto, en este caso con otro tipo de letra, y ademas guardara automaticamente todos los caracteres vacios que usemos, es decir, todos los espacios, saltos de linea o tabulaciones. Simplemente se usara como el tag de parrafo pero en vez de escribir "p" escribiremos "pre".
Aqui se ve un ejemplo: 

              My Bonnie lies over the ocean.
              My Bonnie lies over the sea.
              My Bonnie lies over the ocean.
              Oh, bring back my Bonnie to me.

Como puedes ver el formato cambia considerablemente.

Listas HTML

Las listas HTML son basicamente eso, una lista y puede ser de dos tipos, orderered list (ol) o unordered list (ul).
La sintaxis se veria de la siguiente manera:

<ul>
<li>Coffee </li>
<li>Tea </li>
<li>Milk </li>
</ul>
<ol>
<li>Coffee </li>
<li>Tea </li>
<li>Milk </li>
</ol>

El tag ol y ul son aquellos que distinguiran entre una ordered list y una unordered list.
Por otro lado tenemos el tag li que se incluye en cualquiera de los dos tipos de listas, ya que es el tag con el que se representa cada elemento de estas listas.
La anterior sintaxis se veria de la siguiente forma:

  • Tea
  • Milk
  1. Coffee
  2. Tea
  3. Milk

Distincion de mayusculas

Una cosa a tener en cuenta es que los tags HTML no tienen distincion de mayusculas, es decir, puedes poner los tags en mayusculas o en minusculas, pero se recomienda hacerlo en minusculas para seguir un mismo estandar en todas las paginas.

Atributos HTML

Todos los elementos HTML pueden tener atributos los cuales aportan informacion extra sobre dicho elemento. Estos atributos se suelen poner en el tag de apertura como por ejemplo se puede ver en los tag de enlaces o de imagen.
Suelen tener un nombre y un valor que se veria asi: name="value" .

El atributo style se usa para estilizar un elemento por ejemplo cambiando su color, fuente, tamaño etc.

Tags de formateo de texto

Tags <b> y <strong>

En este caso ambos tags mostraran el texto que contengan en negrita, pero semanticamente el tag b no tendra ningun significado, y el tag strong tendra cierta importancia semanticamente hablando.

Tags <i> y <em>

Estamos ante el mismo caso que el anterior, el tag i simplemente muestra el texto en cursiva, y el tag em aparte de mostrar el texto en cursiva, lo emfatiza y le añade cierto significado semantico.

Tag <small>

Basicamente muestra el texto mas pequeño, como se ve aqui. Ese "aqui" esta dentro de un tag small.

Tag <mark>

Este tag resaltara lo que haya dentro del tag de la siguiente forma: resaltado.

Tag <del>

Este tag definira texto eliminado, es decir, tachara el texto dentro del tag. tachado

Tag <ins>

Este tag definira texto añadido, es decir, subrayado. subrayado.

Tag <sub>

Este tag definira texto como algo mas pequeño un poco como por abajo. subscripted.

Tag <sup>

Este tag basicamente funciona como el anterior pero pone el texto por encima. superscripted.

Citas, tag <q>

Este tag define una cita, y mete el texto entre unas comillas no seleccionables.

Citas, tag <blockquote>

Este tag define una seccion citada desde otro sitio, aqui puedes ver la sintaxis:

<blockquote cite="http://www.worldwildlife.org/who/index.html">texto</blockquote>

Estilos HTML - CSS

CSS es el acronimo de Cascading Style Sheets, es decir, plantillas de estilo en cascada.
Esto indica como se tiene que mostrar el documento HTML en el dispositivo de salida.
Puede ser aplicado al documento de 3 formas diferentes:

  • Inline - Usando el atributo style en los elementos HTML.
  • Internal - Usando un tag style en el head.
  • External - Usando un archivo CSS externo.

Inline CSS

Se utiliza para aplicar un estilo a un unico elemento usando el atributo style como ya hemos hecho anteriormente

Internal CSS

Se usa para definir un estilo para una pagina HTML entera, definiendolo en el tag head de la pagina en un tag style.

External CSS

Se definen los estilos de los elementos HTML en un archivo CSS externo que luego se enlaza con el documento HTML aplicando a cada elemento el estilo correspondiente definido en el documento HTML.
El enlace entre el HTML y el documento CSS se realiza en el head mediante un tag link cuya sintaxis es la siguiente:

El elemento div (BLOQUE)

El elemento div es un elemento que se usa para tener otros elementos en un mismo bloque. De esta forma puedes por ejemplo aplicar css a cierto bloque de codigo sin tener que usar clases.
